Sesión 1: Modelo de Investigación


Este curso plantea apoyarte durante el proceso de investigación interdisciplinaria, es decir, donde profesionales de diferentes especialidades pueden intervenir. Así mismo, este apoyo rádica en brindarte herramientas claves para los diferentes momentos del proceso.

Cada herramienta tiene una función determinada, sin embargo, a lo largo del curso, veremos cómo podemos integrarlas de una manera sencilla.
